## 摘要
现有记忆增强的LLM智能体仅将过往经验存储在提示空间中，以文本摘要或检索段落的形式，同时在整个展开过程中保持模型参数冻结。这类智能体可以\emph{查找}它们见过的东西，但无法从中\emph{学习}：其策略不因经验而改变，任何从上下文中丢弃的信息将永久丢失。我们提出了\texttt{TMEM}，一个自我演化的参数化记忆框架，其中智能体不仅将历史压缩为显式记忆，还通过轻量级在线更新将蒸馏后的监督信息吸收到快速LoRA权重$Δ_t$中，从而在单个情节内真正改变其未来行为。我们将其形式化为具有快速权重展开动态的智能决策过程：动作从$π_{θ_0+Δ_t}$中采样，而提取动作产生用于更新后续决策的$Δ_t$的监督信息。这一视角使得提取策略可以直接通过强化学习进行优化：训练$θ_0$不仅改进了任务动作，还改进了用于在线LoRA适应的数据质量。我们进一步提出了基于SVD的LoRA子空间初始化以加速在线收敛。在LoCoMo、LongMemEval-S、多目标搜索和CL-Bench上的实验表明，\texttt{TMEM}在不同模型规模上始终优于基于摘要和基于检索的基线。

## Abstract
Existing memory-augmented LLM agents store past experience exclusively in prompt space, as textual summaries or retrieved passages, while keeping model parameters frozen throughout a rollout. Such agents can \emph{look up} what they have seen but cannot \emph{learn from} it: their policy is unchanged by experience, and any information dropped from the context is permanently lost. We introduce \texttt{TMEM}, a self-evolving parametric memory framework in which the agent not only compresses history into explicit memory but also absorbs distilled supervision into fast LoRA weights $Δ_t$ via lightweight online updates, genuinely altering its future behavior within a single episode. We formalize this as an agentic decision process with fast-weight rollout dynamics: actions are sampled from $π_{θ_0+Δ_t}$, while extraction actions produce supervision that updates $Δ_t$ for subsequent decisions. This view makes the extraction policy directly optimizable by RL: training $θ_0$ improves not only task actions but also the quality of the data used for online LoRA adaptation. We further propose SVD-based initialization of the LoRA subspace to accelerate online convergence. Experiments on LoCoMo, LongMemEval-S, multi-objective search, and CL-Bench show that \texttt{TMEM} consistently outperforms summary-based and retrieval-based baselines across different model scales.

## 1. 论文的核心问题与整体含义（研究动机和背景）

现有的大语言模型（LLM）智能体在处理长周期任务时（如多轮对话、扩展搜索轨迹、迭代工具使用等），主要依赖两种记忆机制：
- **摘要式记忆**：将历史压缩为文本摘要，存储在提示（prompt）空间中。
- **检索式记忆**：将历史存储在外部分索引中，在需要时检索相关片段。

这些方法都**不改变模型参数**，因此智能体只能“查阅”历史，但无法从经验中真正**学习**：一旦信息被丢弃或未检索到，后续决策就无法利用。这导致智能体的策略在整个情节（episode）内是固定的，无法自我演化。

本文提出 **TMEM（Self-Evolving Parametric Memory）** 框架，核心思想是让智能体在测试时通过**轻量级在线 LoRA 更新**将蒸馏后的监督信息写入模型参数（快速权重 $\Delta_t$），从而在单个情节内真正改变未来行为，使记忆不仅仅是被动存储，而是主动影响策略。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 2.1 核心思想
- 将智能体决策过程形式化为**快速权重展开动态（fast-weight rollout dynamics）**：
  - 工作上下文 $h_t$、显式记忆 $m_t$、快速参数化记忆 $\Delta_t$（LoRA 权重）三者共同构成智能体状态。
  -动作从自适应策略 $\pi_{\theta_0 + \Delta_t}$ 中采样，其中 $\theta_0$ 是冻结的基础参数，$\Delta_t$ 在情节内通过在线更新变化。
- 当上下文预算 $L_{\max}$ 达到上限时，触发记忆写入模式：使用记忆写入提示 $d$，从当前会话中提取**有根据的 QA 对**作为监督信号，然后执行轻量级在线 SFT 更新 $\Delta_t \to \Delta_{t+1}$。
- 这样，蒸馏后的经验通过快速权重影响后续推理，而不需要重复将相同证据插入提示。

### 2.2 关键技术细节
#### (1) 快速权重展开动态
- 定义记忆触发索引 $\{t_i\}$，当上下文长度 $|\ell(h_{t_i}) + \ell(m_{t_i})| > L_{\max}$ 时触发。
- 在两次触发之间，智能体执行普通交互（任务动作/工具调用）。
- 在触发时刻，智能体以记忆写入提示 $d$ 生成提取动作 $a_{t_i}$（结构化为 QA 对），然后通过在线更新算子 $\mathcal{T}$ 将监督写入 $\Delta_{t_{i+1}}$，同时清空工作上下文（可选）。

#### (2) 基于 SVD 的 LoRA 子空间初始化
- 标准 LoRA 随机初始化 $A$ 和 $B$，导致在线更新时需同时搜索子空间和系数，在小步数下效率低。
- **TMEM 初始化**：对预训练权重 $W$ 进行 SVD 分解 $W = U\Sigma V^\top$，取前 $r$ 个右奇异向量和奇异值，设：
  - $B_0 = 0$，
  - $A_0 = \Sigma_r V_r^\top$（固定$A_0$，仅更新 $B$）。
- **优势**：
  - 初始 $\Delta_0 = 0$，策略与基础模型一致；
  - $A_0$ 已张成最高能量的秩 $r$ 子空间，在线更新只需学习系数；
  - 定理 1 证明：SVD 初始化的期望近似误差不大于随机高斯初始化（严格更小当谱对齐系数 $\rho_r > r/d_{in}$）。

#### (3) 基于策略梯度优化（RL）的基参数训练
- 优化目标：最大化期望奖励 $J(\theta_0) = \mathbb{E}_{\tau \sim p_{\theta_0,\mathcal{T},P_{\text{env}}}}[R(\tau)]$。
- 停止梯度通过在线更新算子 $\mathcal{T}$，使得梯度仅传播到产生监督的提取动作和普通动作上。
- 得到的策略梯度估计：对每个段内的普通动作和边界动作分别计算对数概率梯度。
- 通过 RL 训练，$\theta_0$ 学会产生更好的提取监督，使得后续在线 LoRA 更新更有用。

5. **冻结 SVD 子空间几乎不影响性能**，但减少在线更新参数，是更高效的选择。
6. **存在最优上下文预算**：过小或过大的 $L_{\max}$ 都会降低性能。
