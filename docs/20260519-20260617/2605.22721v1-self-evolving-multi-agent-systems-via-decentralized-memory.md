## 摘要
自演化多智能体系统（MAS）已成为一种有前景的途径，使LLM智能体能够从经验中持续改进，并以持久记忆为基础。然而，现有设计几乎都采用跨智能体共享的集中式仓库，这带来了通信和协调开销，引发了隐私问题，并削弱了智能体的多样性。我们提出DecentMem，一种去中心化记忆框架，其中每个智能体维护自己的双池记忆——一个利用池，包含整合的过去轨迹；一个探索池，包含LLM生成的未见过上下文的候选方案。这两个池基于来自LLM-as-a-judge的阶段性反馈进行在线重新加权。理论上，我们证明该设计保证了解决方案空间的全局可达性，并实现了$O(\log T)$的累积遗憾，与随机老虎机下界在常数范围内匹配。在实践中，跨三个MAS框架（AutoGen、DyLAN、AgentNet）、三个Qwen3骨干（4B/8B/14B）、两个Gemma4骨干（E2B/E4B）以及涵盖数学、代码、问答和具身任务的五个基准测试，DecentMem相比最强集中式记忆基线平均准确率提升高达23.8%，相比无记忆基线提升高达52.5%，同时token使用量减少高达49%。

## Abstract
Self-evolving multi-agent systems (MAS) have emerged as a promising route to LLM agents that continually improve from experience, with persistent memory at their foundation. However, existing designs almost exclusively adopt a centralized repository shared across agents, incurring communication and coordination overhead, raising privacy concerns, and collapsing agent diversity. We propose DecentMem, a decentralized memory framework in which each agent maintains its own dual-pool memory -- an exploitation pool of consolidated past trajectories and an exploration pool of LLM-generated candidates for unseen contexts. The two pools are reweighted online based on stage-wise feedback from an LLM-as-a-judge. Theoretically, we prove that this design guarantees global reachability of the solution space and achieves $O(\log T)$ cumulative regret, matching the stochastic bandit lower bound up to constants. In practice, across three MAS frameworks (AutoGen, DyLAN, AgentNet), three Qwen3 backbones (4B/8B/14B), two Gemma4 backbones (E2B/E4B) and five benchmarks spanning math, code, QA, and embodied tasks, DecentMem improves average accuracy by up to 23.8% over the strongest centralized memory baseline and by up to 52.5% over the no-memory baseline, while reducing token usage by up to 49%.

## 1. 核心问题与整体含义（研究动机和背景）

- **研究动机**：基于LLM的多智能体系统（MAS）能够通过跨任务积累经验自我进化，记忆机制是核心。但现有设计几乎全部采用**集中式记忆**（一个全局共享仓库，所有智能体读写），这带来三大问题：
  - 高通信与协调开销；
  - 隐私和安全风险；
  - 智能体行为趋同、多样性丧失——破坏了“角色互补专门化”这一多智能体协作的核心价值。
- **整体含义**：需要一种**去中心化记忆**范式，让每个智能体保留私有经验，同时保持独立探索能力，以兼顾性能、效率、隐私和多样性。

## 2. 方法论：核心思想、关键技术细节

**核心思想**：DecentMem 为每个智能体维护一个私有的**双池记忆**结构——利用池（E-pool）和探索池（X-pool），并通过在线路由动态平衡二者。

- **利用池（E-pool）**：存储来自过去任务的已整合轨迹（包含动作序列、协调轨迹、自我评注），通过**相似性检索**（基于嵌入的语义匹配）重用成功策略，实现局部游走。
- **探索池（X-pool）**：对于未见任务或新上下文，由LLM直接生成全新候选记忆，注入新鲜概率质量，避免陷入局部最优。
- **在线路由器**：根据阶段式反馈（由LLM-as-a-judge评估每个阶段的正确性、分配质量、连贯性、集成质量）动态调整两个池的权重。若当前阶段所用池带来奖励提升，则增加其权重；否则衰减。具体更新规则：
  - 若利用池被选择且阶段得分提升，则利用权重增加；若失败则衰减。
  - 若探索池被选择且阶段得分提升，则利用权重衰减（防止过度依赖过去）；若失败则增加利用权重。
- **记忆合并**：任务完成后，探索池中的所有记忆被合并到利用池，探索池重置。

**理论保证**：将搜索建模为带启发式跳转的随机游走，证明：
- **全局可达性**（Theorem 1）：只要探索池权重>0且LLM先验有完全支持，转移矩阵严格正，马尔可夫链不可约且非周期，每个智能体不会永久困于局部最优。
- **对数累积遗憾**（Theorem 2）：在线路由策略的累积遗憾为 $O(\log T)$，与随机老虎机下界 $\Omega(\log T)$ 阶最优，且远优于任何固定路由策略（线性遗憾）。

## 6. 主要结论与发现

- **准确率提升**：DecentMem平均相对提升8.6%于最强集中基线（G-Memory），最高达23.8%（Qwen3-4B+AgentNet）；相比无记忆基线最高提升52.5%。
- **token节省**：减少高达49%的token使用（平均节省43%），且性能更高。
- **自演化更快**：在MBPP-Plus上，DecentMem达到高性能所需任务数远少于基线（约2.5倍快）。
- **优势随协调随机性增大而扩大**：从AutoGen到AgentNet，相对提升从~2.7%增至~23.1%（Qwen3），验证去中心化在随机协作中更关键。
- **消融证实在线路由的必要性**：固定策略中最好的“仅利用”也失去3-7%准确率，而“仅探索”在部分任务（如BBH）崩溃。

## 7. 优点

1. **方法创新**：首次提出针对MAS的去中心化双池记忆框架，在线路由由阶段反馈驱动，每个智能体独立平衡利用与探索。
2. **理论坚实**：给出全局可达性和对数遗憾上界，达到阶最优，为记忆策略提供理论保障。
3. **实验覆盖广**：跨不同MAS框架（3种）、LLM规模（4B~14B）和架构（稠密/ MoE）、多个域，验证普适性。
4. **效率显著**：相比集中式不仅更准，还更省token，对实际部署有直接价值。
5. **隐私友好**：经验不跨智能体共享，降低隐私风险，天然支持差分私有等扩展。

## 8. 不足与局限
