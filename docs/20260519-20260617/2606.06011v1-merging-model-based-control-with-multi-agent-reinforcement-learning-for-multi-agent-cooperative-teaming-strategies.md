## 摘要
在这项工作中，我们提出了一个框架，将多智能体强化学习与基于模型的控制相结合，以在多智能体协作任务中实现安全、动态可行的动作。多智能体强化学习具有从长规划视野中的离散不可微奖励中学习多智能体团队协作策略的优势。模型预测控制具有鲁棒性，并在快速重规划框架中为短视野提供安全、动态可行的动作。我们提出了一种扩展演员-评论家模型预测控制用于多智能体强化学习的算法，称为多智能体演员-评论家模型预测控制（MA-AC-MPC）。我们通过将其应用于多智能体追逃场景来展示该算法的能力。具体而言，我们比较了使用MA-AC-MPC模型和多层感知器模型（MA-AC-MLP）的逃避者团队策略。追击者团队使用增强比例导引，因其被公认为先进的对抗控制律。我们还提供了一个异构环境的示例，其中无人机和全向轮式机器人合作以实现可重复且成功的着陆，MA-AC-MPC在硬件上的成功率为100%，而MA-AC-MLP为60%。我们在硬件上证明了所提出的MA-AC-MPC算法在两种环境中的鲁棒性。

## Abstract
In this work, we propose a framework that combines multi-agent reinforcement learning (MARL) with model-based control to achieve safe, dynamically feasible actions in cooperative multi-agent tasks. Multi-agent reinforcement learning provides the advantage of learning cooperative policies for multi-agent teams from discrete non-differentiable rewards in a long planning horizon. Model-predictive control is robust and offers safe, dynamically feasible actions in a fast replanning framework for short horizons. We propose an algorithm that extends actor-critic model predictive control for MARL which we refer to as multi-agent actor-critic model predictive control (MA-AC-MPC). We demonstrate the capabilities of this algorithm by applying it to a multi-agent pursuit-evasion scenario. Specifically, we compare the evader team's strategy using the MA-AC-MPC model and a multi-layer perceptron model (MA-AC-MLP). The pursuer team uses augmented proportional navigation as it is accepted as an advanced adversarial control law. We also provide an example with a heterogeneous environment where a drone and omni-wheeled rover cooperate to achieve repeatable and successful landing with 100% success rate in hardware for MA-AC-MPC compared to 60% for MA-AC-MLP. We demonstrate the robustness of the proposed MA-AC-MPC algorithm in hardware for both environments.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究动机**：多智能体强化学习（MARL）能从长规划视野的非连续、不可微奖励中学习协同策略，但其输出动作可能违反物理约束、缺乏鲁棒性，尤其在安全关键任务中不可靠。模型预测控制（MPC）利用已知物理模型和约束，能快速在线重规划并保证动作的动态可行性与安全性，但其目标函数需人为设计，难以应对复杂长期协作任务。
- **核心问题**：如何将MARL的长期协同能力与MPC的短时安全、动态可行性相结合，实现多智能体团队的安全高效协作。
- **整体含义**：提出一种融合框架——多智能体演员-评论家模型预测控制（MA-AC-MPC），将MPC作为可微分的动作生成层嵌入MARL的演员网络，从而同时获得长期奖励优化和实时约束满足，提升策略的鲁棒性和迁移能力。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 2.1 核心思想
- 采用“集中训练-分散执行”（CTDE）范式，每个智能体拥有自己的演员网络，但训练时共享全局状态值函数。
- 演员网络分为两层：第一层为神经网络（成本网络），输出MPC所需的参数（参考轨迹、权重矩阵等）；第二层为可微分MPC求解器（基于`leap-c` + `acados`），根据当前状态和成本参数生成控制动作。
- 评论家网络估计全局状态下的值函数，用于计算优势函数，指导演员更新。

### 2.2 关键技术细节
- **算法基础**：基于多智能体近端策略优化（MAPPO），使用广义优势估计（GAE）。
- **MPC公式**：采用非线性MPC（NMPC），状态方程 `x_{k+1}=f_k(x_k,u_k;θ_MPC)`，目标函数为二次型或最小二乘形式，包含状态和控制约束。可学习参数包括阶段成本参考、终端成本参考、权重矩阵等。
- **可微分MPC实现**：利用开源框架`leap-c`，它提供基于`acados`的敏感性传播接口，能高效计算MPC解对参数的梯度，并兼容PyTorch/JAX。
- **死亡掩蔽**：对失活智能体（如相撞、出界）的观测和状态进行零向量掩码，并添加二进制存活标志和one-hot ID，防止价值网络输入分布偏移。
- **共享策略**：同构智能体（如追逃中的evader）共享同一组演员网络参数，提高样本效率。

### 2.3 公式与算法流程（文字说明）
1. 每个智能体i在当前时间步获得局部观测`o^{(i)}`。
2. 成本网络`π^{(i)}_θ`输出MPC参数（参考点、权重等）。
3. 将智能体当前状态`x^{(i)}`和成本参数输入MPC求解器，求解有限时域最优控制问题，得到控制动作`u^{(i)}`。
4. 所有智能体执行联合动作，环境转移到下一状态，并返回团队奖励。
5. 收集一段时间的轨迹，使用GAE计算优势估计。
6. 更新评论家网络（均方误差损失）和演员网络（裁剪代理目标 + 熵正则），梯度通过可微分MPC反向传播。
7. 循环直至收敛。
